# Change: Add Chinese New Year Customs QA System

## Why
There is a need for an intelligent QA system that can answer questions about Chinese New Year customs and traditions in an accurate and engaging way. This system will help users learn more about Chinese culture and traditions.

## What Changes
- Add a knowledge base of over 50 Chinese New Year customs and traditions
- Implement a RAG (Retrieval-Augmented Generation) workflow using TRAE
- Support multi-turn dialogues for follow-up questions
- Generate colloquial answers that are easy to understand

## Impact
- Affected specs: New capability for Chinese New Year customs QA
- Affected code: New files for knowledge base, RAG workflow, and dialogue management
